Assignment Descriptions

  • Assignment 1 - Creating Your First Web Page
  • This assignment was the creation of the first website which would also function as the index for future assignments that would be assigned in GUI Programming I as well as showing some basic info on Kevin Huang
  • Assignment 2 - Style a Site With External CSS
  • This assignment was an excercise in learning how to use external CSS style sheets to replicate an example website of a resort called Pacific Trails Resort. It tested knowledge of web page styling as well as whether or not you could see some of the sneaky stylings that were going on in the background.
  • Assignment 3 - Styling Your First Web Page with CSS
  • This assignment was a combination of the previous two assignments. We took the original ugly appearance of a purely html web page that I first made and used the external CSS techniques on it that were learned and reinforced when recreating the Pacific Trails Resort website.
